## 公司简介

湛江伟力机电设备有限公司（以下简称：公司）于2007年在广东省湛江市创基立业，注册资金2000万元，具备精湛的服务技术和丰富的销售经验，专业从事发动机维修，发电机维修，发电机组安装、维修及租赁，发电机组附属产品的销售及维护等。

公司经营5～3000KW的动力及发电产品，主营国内外主流品牌，种类包含：陆用柴油发电机组、船用柴油发电机组、低中高压发电机组、燃气发电机组及变频变压电源等，可承接全自动配电项目，形成了从技术咨询、设备供应、安装调试、维修、零件供应、培训等全套服务的经营特色。

公司服务于石油、化工、通讯、船舶、军工、矿山、运输、环保、食品等行业，范围已经覆盖了全国各省份及大中城市。多年来，公司以追求高品质、可靠、专业为运营目标，服务至上是我们的宗旨，致力于以优质产品和个性化服务满足客户需求，以信誉树立品牌，以实力见证未来。

公司取得安全生产许可证，已通过ISO9001质量管理体系、ISO14001环境管理体系、ISO45001职业健康安全管理体系的认证，并取得康明斯发动机授权经销商（修理级服务），利莱森玛发电机授权经销商，2024年被认定为高新技术企业、科技型中小企业、创新型中小企业、专精特新中小企业，连续三年度纳税信用A级，荣获“湛江市诚信经营示范单位”证书。

公司座落于湛江市麻章区瑞云路之侧，占地面积6000多平方米，交通便利。公司设有生产车间、测试中心、维修车间及零件仓库，设施完善。公司拥有一支强大且训练有素的技术服务队伍，35名服务人员中有多人取得发电机组行业厂家技能培训证书，如康明斯发动机。凭借专业技术和制造经验的积累以及优良的企业文化，团结、创新的人才团队和科学的管理模式，遵循“诚信为本，服务至上”的宗旨，竭诚为客户提供优良的产品和周到的服务。在华南中心选址成立分公司，位于广州市南沙区南沙大道，占地面积4000平方米，服务人员15名。从而进一步扩大销售区域，服务范围更广，响应速度更快，更高效地为各地用户提供优质的产品和服务。公司设有24小时服务热线电话，及时为客户提供全面的保障。
